    How To Play?

    Factions

    Getting started is one of the most difficult things. You have to build a base and gain trustworthey teammates. First of all you start by creating a faction by typing the following comand: "/f create (faction name)". Secondly you'll have to explore the foreign lands and when you think it's save enough you can set your bace on that location by typing the following: "/f sethome" From here you can build a base. For more advanced features, type in "/f help" for more commands in the game.

    Auctions

    Auctions are most commonly used to sell valuable items. You can set a start price and people can bid on the item. We recommend you to use this command wisely. To start an aution, simply type in the following: "/auction start (item) (count) (start_price)". To stop your auction, simply type the following: "/auction stop". To set an auction only valid for a certian time, type the following: "/aution time (item) (count) (start_price) (time)". To bid on an auction, simply type the following: "/auction bid (player) (price)".

    PvP Arena

    Most commonly players battle in the arena when they're geared up with strong armor. Always make sure you have protection before entering the battle arena. To warp to the PvP arena, simply type the following: "/pvp" or tap on the PvP Arena slapper in the lobby.

    Shops

    Most commonly shops are there to so you can buy your needs. You can also craft them, but there is certian items in the shop that can not be mined or crafted. You'll need money to buy items from the shop. To get to the shops, simply type the following command: "/shop" or tap on the Shop slapper in the lobby.

    Mines

    Mines are used to mine materials and sell them to gain an mount of money. The money are used to buy items in the shop. You can also use the mined materials for your faction or fighting in the PvP arena. To go to the mines, simply type the following command in the server: "/mine" or tap on the slapper located in the server lobby.

    1vs1

    1v1 is almost the same as PvP Arena, except you go one on one. You fight against the player and maybe win some money, who knows? To enter a 1v1 challenge, simply type the following command: "/1v1" or tap on the slapper located in the server lobby.
